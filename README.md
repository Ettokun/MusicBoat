# MusicBoat
Discord music bot using **[discord-player](https://npmjs.com/package/discord-player)**

# How to use?
- Download the project
- Rename `config.example.js` to `config.js`
- Fill the correct details in `config.js`
- Download and install FFmpeg
- Run `node index.js`
